# Demo

This demo intentionally uses synthetic public content.

Expected flow:

1. Generate visual layers with Image Split from `image-split/examples/demo/input.png` and `image-split/examples/demo/elements.json`.
2. Copy the generated page folder to `image-ppt-king/examples/demo/visual-layers/slide-01`.
3. Build the PPTX using `text-layer.json`.
