# QA Rubric

A reconstruction is acceptable only when:

- The PPTX opens and has the expected slide count.
- Main titles, card text, bullets, captions, and page numbers are native editable text.
- Text boxes are transparent and do not patch over missing visual shapes.
- Visual assets are selectable objects, not one flattened screenshot.
- Rendered previews have no obvious missing icons, labels, lines, cards, or duplicated text.
- QA reports document remaining limitations.

For detailed gates, see `skills/image-ppt-king/references/acceptance-rubric.md`.
